About EnerPredict AI

EnerPredict AI is a smart energy analytics platform developed as part of the AI for Renewable Energy mini-project. It combines machine learning, appliance-level modeling, and intuitive web dashboards to help users understand and optimize their household energy consumption.

What the system does

  • Predicts monthly electricity consumption using trained ML models (XGBoost, Random Forest, Linear Regression).
  • Analyzes appliance-wise energy usage to identify wastage or inefficiency.
  • Provides solar sizing recommendations based on user demand and panel specs.
  • Validates inputs intelligently using a hybrid dataset-driven + rule-based system.
  • Visualizes predictions and usage trends using interactive dashboards.

Machine Learning

The backend uses trained models on a cleaned household energy dataset. StandardScaler preprocessing and a strict feature-order pipeline ensure accurate predictions. XGBoost typically delivers the highest accuracy, while Linear Regression is the fastest.

Key Features

  • Interactive prediction interface with auto-fill, tooltips, and smart suggestions.
  • Hybrid validation engine using statistical percentiles and domain rules.
  • Local storage for appliance lists and prediction history.
  • Solar calculator and idle/wasted energy insights.
  • Modern UI built with HTML, CSS, JavaScript, and Chart.js.
  • REST API backend built in Python + Flask.

Project Motivation

The goal of this project is to demonstrate how AI can be applied in the renewable energy domain to improve household energy efficiency, reduce wastage, and make solar deployment more informed and accessible.
